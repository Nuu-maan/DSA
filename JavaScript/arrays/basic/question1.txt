Reverse an Array
Source: https://www.geeksforgeeks.org/problems/reverse-an-array/1

Given an array A of size N, return the reverse of it.

Example 1:
Input:
N = 4
A[] = [1, 2, 3, 4]
Output: [4, 3, 2, 1]

Example 2:
Input:
N = 5
A[] = [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]

Example 3:
Input:
N = 1
A[] = [10]
Output: [10]

Constraints:
- 1 <= N <= 10^5
- 1 <= A[i] <= 10^6

Note: Implement multiple approaches showcasing JavaScript's array methods and ES6+ features.
