Check if a number is prime

Given a positive integer n, check if it is a prime number.

A prime number is a natural number greater than 1 that has no positive divisors other than 1 and itself.

Examples:
Input: n = 7
Output: True
Explanation: 7 is a prime number (divisors: 1, 7)

Input: n = 4
Output: False
Explanation: 4 is not a prime number (divisors: 1, 2, 4)

Input: n = 1
Output: False
Explanation: 1 is not considered a prime number

Input: n = 2
Output: True
Explanation: 2 is a prime number (divisors: 1, 2)

Constraints:
- 1 <= n <= 10^9
