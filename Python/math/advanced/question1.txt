Q1: Given an integer n, return the nth digit of the infinite integer sequence [1,2,3,4,5,6,...].
Source: LeetCode 400 - Nth Digit

The sequence is formed by concatenating all positive integers: 123456789101112...

Sample Input:
n = 11

Sample Output:
0

Explanation:
The 11th digit in the sequence is '0' (from the number 10).
