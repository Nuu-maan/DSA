Q3: Given an array of integers, return the sum of all XOR totals for every subset.
Source: LeetCode 1863 - Sum of All Subset XOR Totals

Given an array nums, return the sum of all XOR totals for every subset of nums.

Sample Input:
nums = [1, 3]

Sample Output:
6

Explanation:
The subsets are [ ], [1], [3], [1,3]. Their XOR totals are 0, 1, 3, 2. The sum is 0 + 1 + 3 + 2 = 6.
