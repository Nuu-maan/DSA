Q3: Given an integer n, return the count of all numbers with unique digits in the range [0, 10^n).
Source: LeetCode 357 - Count Numbers with Unique Digits

Given a non-negative integer n, count all numbers with unique digits, x, where 0 <= x < 10^n.

Sample Input:
n = 2

Sample Output:
91

Explanation:
There are 91 numbers with unique digits between 0 and 99 (inclusive).
